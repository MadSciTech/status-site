On Monday, July 17, 2017, starting at 9:40 PM (CDT) our primary web server started to experience network connectivity loss. At 9:53 PM (CDT) we received confirmation that there is a network outage currently in process. At 9:55 PM (CDT) our Managed Web Services came back online, slowly at first.
